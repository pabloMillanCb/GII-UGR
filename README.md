# Practicas varias del Grado en Ingenieria Informatica UGR
Proyectos varios de algunas asignaturas de la carrera de informatica en la ugr, cursada en la promocion 2019-2023. El repositorio cuenta con las asignaturas:
- Estructuras de Datos (ED)
- Estructura de Computadores (EC)
- Programacion Orientada a Objetos (POO)
- Metodologia de la Programacion (MP)
- Informatica grafica (IG)
- Metaheuristicas
- Sistemas Informaticos Basados en Web (SIBW)
- Desarrollo Basado en Agentes (DBA, la mejor asignatura, y el mejor profesor Luis <3)
